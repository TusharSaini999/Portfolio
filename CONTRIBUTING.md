# Contributing

Thank you for considering a contribution to this portfolio.

## How to Contribute

1. Fork the repository.
2. Create a feature branch from `main`.
3. Make your changes.
4. Test the app locally.
5. Open a pull request with a clear description of the change.

## Development Setup

This project uses Vite for the frontend and a separate function package for the contact email service.

### Web app

```bash
cd Web
npm install
npm run dev
```

### Contact email function

```bash
cd Function/contact-email
npm install
```

## Before Opening a PR

- Keep changes focused and minimal.
- Follow the existing code style.
- Check that the portfolio still builds and runs correctly.
- Avoid committing secrets or environment files.

## Reporting Issues

If you find a bug or have a feature request, open an issue and include as much relevant context as possible.
