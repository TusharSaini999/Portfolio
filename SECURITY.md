# Security Policy

## Supported Versions

Security updates are maintained for the latest version of the portfolio source code.

## Reporting a Vulnerability

If you discover a security issue, do not open a public issue. Instead, report it privately to the repository owner using the contact information listed in the project README.

Please include:

- A short description of the issue
- The affected file or feature
- Steps to reproduce, if applicable
- Any suggested mitigation, if known

We will review reports promptly and coordinate a fix as needed.
